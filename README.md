# Meituan Delivery Data Analysis（外卖数据分析项目）

## 📌 项目背景
随着外卖行业竞争加剧，平台需要通过数据分析提升用户转化率与复购率。本项目基于外卖平台订单与用户行为数据，分析用户从浏览到下单的关键路径，识别流失环节，并提出优化建议。

## 🎯 项目目标
- 分析用户下单转化路径（浏览 → 加购 → 下单）
- 识别影响转化率的关键因素
- 分析用户留存与复购情况
- 提出提升平台运营效果的优化策略

## 📊 数据说明
- 数据来源：模拟外卖平台数据
- 数据内容包括：
  - 用户ID
  - 订单ID
  - 下单时间
  - 订单金额
  - 商家评分
  - 是否复购

## 🧠 分析思路
1. **数据清洗**
   - 去除缺失值与异常值
   - 统一时间格式

2. **用户行为分析**
   - 分析用户访问与下单行为分布
   - 统计不同时间段订单量

3. **转化漏斗分析**
   - 浏览 → 加购 → 下单转化率计算
   - 找出流失最严重的环节

4. **用户留存分析**
   - 计算新用户复购率
   - 分析用户生命周期价值（LTV）

## 🛠 使用工具
- Excel（数据处理与基础分析）
- SQL（数据提取与统计）
- Python（pandas）（数据分析）
- 可视化工具（Power BI / Tableau）

## 📈 核心发现
- 晚间（18:00–21:00）为订单高峰时段
- 浏览到加购转化率较低，存在明显流失
- 新用户复购率偏低，留存能力不足
- 高评分商家订单转化率显著更高

## 🚀 优化建议
- 优化推荐算法，提高用户浏览到加购转化
- 针对新用户推出优惠策略，提高首单后复购率
- 强化高评分商家曝光，提升整体转化率
- 在高峰时段优化配送能力，提升用户体验
